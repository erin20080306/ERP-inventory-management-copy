艾琳 ERP 公司主機手動安裝說明
版本：v1.1.2-manual-macos-safety

這個 ZIP 是「公司主機手動安裝包」。下載並解壓縮後，由使用者自行雙擊安裝檔；安裝程式會建立公司主機、安全背景更新服務，並嘗試下載目前電腦適用的艾琳 ERP 工作站。ZIP 不會重新散布 Docker Desktop。

同一台電腦可以同時安裝：
1. 艾琳 ERP 公司主機
2. 艾琳 ERP 工作站

安裝順序：
1. 下載並解壓縮 Host ZIP。
2. 雙擊 ZIP 根目錄的「安裝艾琳ERP.command」（macOS）或「安裝艾琳ERP.bat」（Windows）。
3. 若尚未安裝 Docker Desktop，安裝程式會開啟 Docker 官方安裝頁並停在原視窗等待。
4. macOS 會依處理器自動提示正確版本：Apple Silicon 選「Mac with Apple silicon」，Intel Mac 選「Mac with Intel chip」。
5. 安裝並開啟 Docker Desktop，第一次啟動請接受 Docker 條款並完成系統權限設定。
6. 回到艾琳 ERP 安裝視窗繼續，不必重新下載 ZIP。
7. 輸入艾琳設計提供的啟用碼。
8. macOS 會優先使用預設網路介面的區網 IPv4；若無法自動判斷，會請你手動輸入，例如 192.168.1.20，不會直接使用 127.0.0.1。
9. 等待畫面顯示「安裝完成」、公司代碼、主機網址、管理員帳號與密碼。
10. 管理後台顯示「公司主機 1/1」後，從桌面開啟「艾琳 ERP」，輸入公司代碼與啟用碼。
11. 往後有功能新版時，ERP 會顯示通知；公司管理者到「系統設定 → 系統更新中心」點「備份並更新」即可，不需重裝主機。

重要：
- macOS／Windows 安裝完成後都會建立桌面「艾琳 ERP」圖示；若工作站暫時下載失敗，可稍後從 ERP 的「桌面版」補裝。
- 公司主機與工作站可裝在同一台電腦，但主機需要 Docker Desktop 保持執行。
- Mac 若更換 Wi-Fi、網路分享器或公司網段，區網 IP 可能改變；其他工作站無法連線時，請先確認主機網址是否仍是目前 IP。
- 更新前會先建立 AES-256-GCM 加密完整備份；新版未通過健康檢查時會切回舊版映像。資料庫 migration 無法只靠映像回滾，因此重要更新後仍需保留加密備份。
- Docker Desktop 是 Docker 公司提供的第三方軟體，授權條款由 Docker 提供。
- 上方若出現 google-cloud-sdk 的 .zshrc 路徑警告，與艾琳 ERP 安裝無關。
- 安裝完成後請立即保存管理員密碼與備份解密金鑰。
